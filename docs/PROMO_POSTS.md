### D. 영문 (해외 네트워크용)

Building Rimvio — a personal Experience OS (not another chat wrapper).

What we ship:
• Pin lived context on a 3D globe (trips, moments, people)
• Plug in / unplug context hubs (e.g. departure airport → flight search)
• Share → Action: shared links land on an action sheet, not a scroll feed

Stack: Next.js · TypeScript · Supabase · Vercel · PWA

Beta feedback welcome on GitHub Issues.
