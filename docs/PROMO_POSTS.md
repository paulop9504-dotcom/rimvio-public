## Reddit r/SideProject (EN)

**Title:** `Show: Rimvio — Experience OS with a 3D globe, plug-in context hubs, Share→Action`

**Body:**

```
I've been building Rimvio — a personal Experience OS (not a chat wrapper).

- Pin "contexts" (trips, moments) on a 3D globe
- Plug in / unplug "hubs" (e.g. link a trip to a departure airport → opens flight search)
- Share → Now flow: shared links land on an action sheet, not a scroll feed

Stack: Next.js, TypeScript, Supabase, Vercel, PWA.

Try: https://rimvio.app
Source: https://github.com/paulop9504-dotcom/rimvio-public

Would love feedback on the globe + hub UX. Issues welcome on GitHub.
```
