# Changelog

## [0.1.0] — 2026-06-15

### Added

- **Context hubs** — plug in / unplug resource hubs on globe contexts (anytime: before or after creating context)
- **Departure airport hubs** — link travel contexts to ICN / GMP / CJJ; tap hub → Google Flights search
- **Globe context hub panel** on pin sheet (+ add, × remove)
- **Draft hub picker** on manual context create form

### Stack

- Next.js · TypeScript · Supabase Auth · Vercel · PWA
